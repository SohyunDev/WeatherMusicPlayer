1. IntroActivity를 만듦.
2. MainActivity에서 기상청 사이트의 동작구 흑석동 동네예보를 통해 날씨정보를 가져옴.
3. 현재 날씨는 동네 예보의 가장 첫번재 날씨를 이용함.
4. 버튼을 누르면 음악을 재생할 수 있음.
5. 날씨는 wfkor를 기준으로 나눔. (맑음, 구름조금, 구름많음, 흐림, 비, 눈/비, 눈)
6. 현재 시간(System.currentTimeMillis())를 이용하여 
2로 나눈 나머지가 0일때, 1일때를 기준으로 랜덤하게 음악을 재생하도록함.
7. 노래는 반복됨
8. 버튼을 다시누르면 노래가 정지됨.
